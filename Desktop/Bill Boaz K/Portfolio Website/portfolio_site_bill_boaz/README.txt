
Portfolio website (multi-page) for Bill Boaz Kogembo
---------------------------------------------------
Files included:
 - index.html, about.html, projects.html, contact.html
 - assets/style.css
 - assets/main.js

How to use:
 1. Download and unzip the folder.
 2. Place the files in the root of your GitHub repo (not inside src/).
 3. Commit & push. In GitHub Pages settings, set Source: Branch 'main' (root).
 4. Your site will be available at https://<username>.github.io/<repo-name>

Customization:
 - Edit the HTML files to update text, links, or contact details.
 - Replace social links in index.html footer.

Notes:
 - The contact form is static (demo). Replace the form action with your email handler or use GitHub Forms/Netlify forms.
